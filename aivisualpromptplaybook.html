Intro

Cara Pakai Ebook Ini

Ebook ini adalah playbook praktis. Hampir semua prompt ditulis dalam bahasa Inggris karena platform AI visual lebih stabil memahami detail visual dalam bahasa tersebut.

  • Ganti placeholder seperti [PRODUCT], [BRAND], [SCENE]
  • Iterasi kecil: ubah lighting, angle, ratio
  • QC manual sebelum dipakai brand
  • Gunakan AI untuk concept & draft, bukan final produksi tanpa review
Chapter 00

Framework Prompt Profesional

Struktur dasar agar output konsisten dan tidak random.

Master Prompt — Image Generation

Universal
Create a [OUTPUT TYPE] of [MAIN SUBJECT] for [BRAND/USE CASE]. Scene: [SCENE]. Composition: [ANGLE/FRAMING]. Lighting: [LIGHTING]. Style: [VISUAL STYLE]. Details to preserve: [IMPORTANT DETAILS]. Mood: [MOOD]. Output: [ASPECT RATIO]. Avoid: [NEGATIVE DETAILS].

Master Prompt — Image to Video

Animate this exact image into a [DURATION]-second video. Preserve subject identity, shape, label, colors, lighting. Only add [MOTION]. Camera movement: [CAMERA MOVE]. Mood: [MOOD]. Avoid morphing, redesign, extra objects.

Master Prompt — Product Ad Video

Create a [DURATION]-second commercial for [PRODUCT]. Shot 1: [OPENING] Shot 2: [INTERACTION] Shot 3: [FINAL PACKSHOT] Camera: [STYLE] Lighting: [LIGHTING] Mood: [BRAND MOOD]

Negative Prompt & QC Checklist

Negative Prompt Library
